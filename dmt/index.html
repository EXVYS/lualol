Command List

Explore the extensive command set available with DMT-Api

!save

Save server configuration including channels, categories, and roles.

!restore

Restore server configuration from a previous save.

!nuke

Full server nuke - delete channels, create new ones, rename server.

!massban

Ban all members from the server instantly.

!massdm

Spam DM all members with a custom message.

!dmspam

Spam DM a specific user with a custom message.

!broadcast

Send a message to all server members.

!steal

Steal Roblox clothing assets instantly.

!tc

Show total channels and server information.

!config

Show current bot configuration.

!set_config

Set bot configuration with key:value pairs.

!shutdown

Stop all commands and requests instantly.
