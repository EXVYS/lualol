Scanner Bot Policy

Usage Guidelines & Prohibited Activities

Introduction

Scanner Bot is a Discord security and analysis tool designed to help server administrators maintain safe communities. By using this bot, you agree to abide by these policies.

Note: Scanner Bot is intended for legitimate security purposes and community management. Misuse will result in permanent banning from the service.

Acceptable Use Policy

Permitted Activities

  • Scanning for legitimate security threats in your own servers
  • Analyzing coding skills for educational purposes
  • Identifying potential malware or malicious content
  • Community management and user assistance

Administrator Responsibilities

  • Inform your community about scanning activities
  • Use scan results responsibly and avoid false accusations
  • Respect user privacy and Discord's Terms of Service
  • Only scan when there's legitimate need or suspicion

Privacy & Data Handling

What We Collect

  • Message content is processed only when commands are used
  • Results are temporarily cached (up to 1 hour) for performance
  • No personal data is permanently stored or shared

Data Usage

  • Content analysis is performed locally when possible
  • AI code rating uses secure API connections
  • We do not sell or transfer any user data

Important: Server administrators must comply with local privacy laws and Discord's guidelines when using scanning tools.

Prohibited Use & Misuse

The following activities are strictly forbidden and will result in immediate action:

Harassment & Targeting

Using scan results to harass, bully, or falsely accuse users without evidence.

Unauthorized Scanning

Scanning servers where you don't have administrative permission or legitimate purpose.

Automated Abuse

Using bots or scripts to automate scanning commands for malicious purposes.

Service Disruption

Attempting to overload, reverse engineer, or disrupt the bot service.

Zero Tolerance Violations

  • Using the bot for doxxing or personal information gathering
  • Coordinated attacks against other servers or users
  • Violating Discord's Terms of Service or community guidelines
  • Any illegal activities using the bot

Consequences of Misuse

Minor Violation

First offense or unintentional misuse

  • Warning issued
  • Temporary command restrictions
  • Education on proper usage

Moderate Violation

Repeat offenses or clear policy violation

  • Extended command restrictions
  • Server-wide usage limitations
  • Formal warning recorded

Severe Violation

Malicious intent or illegal activities

  • Permanent ban from Scanner Bot
  • Reporting to Discord Trust & Safety
  • Legal action if applicable

Reporting Misuse

If you encounter someone misusing Scanner Bot:

  • Collect evidence (screenshots, message links)
  • Report to the bot developer with complete details
  • Include server IDs, user IDs, and timestamps
  • Allow 24-48 hours for investigation

Policy Updates: This policy may be updated periodically. Continued use of Scanner Bot constitutes acceptance of the current policy version.
